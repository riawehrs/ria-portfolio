New T+ink Website

I was hired by a small digital agency to create the front-end for a new T+ink website. My contact for the project was the agency’s CTO/lead developer, who set me up with the Node-based toolset I needed to create Jade templates and define routes with Express (note: the site linked above is a static html version).

I was given mockups for the full desktop site and adapted the page layouts as needed to make them responsive. I had the freedom to make minor visual adjustments as I implemented the design, and chose to alter some types styles and spacing to make them more globally consistent and impose a stronger sense of vertical rhythm. The agency was very happy with the result.
